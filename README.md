# Vixen Labs CX

Structured AI character creator scaffold built with React, Vite, Tailwind, Zustand, Dexie, and a split schema system.

## Run

```bash
npm install
npm run dev
```

## Current Scope

- Split category schema under `public/schema/categories/`
- Model configs under `public/models/`
- Mobile-first UI with swipe navigation
- Prompt assembly, randomization, presets, and placeholder gallery entries
- PWA shell via `vite-plugin-pwa`

## Placeholder Data

Some schema options are intentionally marked with `"placeholder": true` so they can be replaced with final product-approved values later.
